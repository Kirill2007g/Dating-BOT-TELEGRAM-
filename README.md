# Dating Bot 💘

Telegram бот для знакомств / Telegram dating bot

---

## RU

### Функции
- Регистрация с фото, видео или кружочком
- Поиск анкет по городу
- Алгоритм подбора: ±2 года от твоего возраста (90%), случайные анкеты (10%)
- Лайки / дизлайки
- Уведомления о лайках
- Взаимные симпатии с отправкой контактов
- Редактирование профиля (фото, текст, полная перезапись)
- Деактивация анкеты

### Премиум
- Возврат к пропущенной анкете
- Расширенный диапазон поиска

### Стек
Python 3.12 · aiogram 3.7 · SQLite

### Запуск
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
# Создай .env с BOT_TOKEN=твой_токен
python bot.py
```

## EN

### Features
- Registration with photo, video or video note
- Profile search by city
- Matching algorithm: ±2 years from your age (80%), random profiles (20%)
- Like / dislike system
- Like notifications
- Mutual matches with contact sharing
- Profile editing (photo, text, full rewrite)
- Profile deactivation

### Premium
- Undo — return to a skipped profile
- Extended search range

### Stack
Python 3.12 · aiogram 3.7 · SQLite

### Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
# Create .env with BOT_TOKEN=your_token
python bot.py
```
